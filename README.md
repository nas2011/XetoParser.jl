# XetoParser

Julia implemented parser of the [Xeto](https://github.com/Project-Haystack/xeto) language based on the BNF grammar spec in the [language docs](https://github.com/Project-Haystack/xeto/blob/master/src/xeto/doc.xeto/Grammar.md).  

Work in progress. May not successfully parse all edge cases, but has handled pretty complex examples to this point.
